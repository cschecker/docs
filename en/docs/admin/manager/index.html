Manager Settings

info

cloud security checker: super org(top-level organization)is a service that manages sub orgs (Sub-Organization). The super org (top-level organization) can have multiple administrators under it.

Inviting a Manager​

  1. Click the Add Administrator button.

  1. Click Invite to send an invitation email.

    Enter the name and email of the user to be invited.
    Click Add to List + to add the user to the administrator list.
    Click Invite again to send the invitation email.


Invited: The email has been sent.
Activated: The user has received the invitation email and registered the OTP in the service How to register OTP.


 

Deleting a Manager​

Click Settings (three-dot icon) and then click the Remove Admin button.
The re-invite button is only activated when the status is Invited.

Invited: The email has been sent.
Activated: The user has received the invitation email and registered the OTP in the service How to register OTP.
