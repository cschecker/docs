Sub-Organization Management

To manage Sub-Organization, click on the Admin>Sub-Organization Managementpage.

info

cloud security checker: The super org (top-level organization) is a service that can manage sub orgs (Sub-Organization). The super org (top-level organization) can have multiple administrators under it.

Adding, Deleting, and Editing Sub-Organization​

Adding​

  1. Click the Add Sub-Organization Button.

  2. Enter the Sub-Organization Name.

  3. Select the company logo. (196 x 60px)

  4. Click the Add button.

Invited: The email has been sent.
Activated: The user has received the invitation email and registered the OTP in the service How to register OTP.


 

Deleting, Editing​

  1. Click the Edit Sub-Organization Button.

  2. Editing: Click the text input field, modify the Sub-Organization name, and then click Edit to apply changes.
    Deleting: Click the Delete Button.

 

Inviting Users to a Sub-Organization Group​

  1. Click the Invite User Button.

  2. Enter the Name(Real Name)and Email of the user you want to invite.

  3. Set the user's permissions (multiple selections allowed). If you want to invite multiple users, click the Add to List + and enter the user information.

Invited: The email has been sent.
Activated: The user has received the invitation email and registered the OTP in the service How to register OTP.
