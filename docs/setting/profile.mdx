# 프로필

## 프로필 이미지

<img width="400" src={profileImage1} alt="프로필변경 이미지" />

1. CS Checker - 설정 - 내 프로필에서 `프로필 변경`를 클릭합니다.
2. 원하는 이미지를 선택 후 변경합니다.

## 비밀번호 재설정

1. CS Checker - 설정 - 내 프로필 에서 `비밀번호 재설정`를 클릭합니다.

    <img width="200" src={profileImage2} alt="비밀번호재설정 이미지" />

2. 기존 비밀번호를 입력합니다.

    <img width="400" src={profileImage3} alt="비밀번호재설정 이미지" />

3. 새 비밀번호를 입력합니다.
4. 확인을 위해 다시한번 새 비밀번호를 입력합니다.
5. `바꾸기` 버튼을 눌러 완료합니다.
