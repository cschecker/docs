# 프로필

## 프로필 이미지

<img width="400" src={profileImage} alt="프로필변경 이미지" />

1. CS Checker - 설정 - 내 프로필 에서 `프로필변경`를 클릭합니다.
2. 원하는 이미지를 선택 후 변경합니다.

## 비밀번호 재설정

<img width="400" src={resetPw} alt="비밀번호재설정 이미지" />

1. CS Checker - 설정 - 내 프로필 에서 `비밀번호 재설정`를 클릭합니다.
2. 기존 비밀번호를 입력합니다.
3. 새 비밀번호를 입력합니다.
4. 확인을 위해 다시한번 새 비밀번호를 입력합니다.
5. `바꾸기` 버튼을 눌러 완료합니다.

## OTP 등록

1. 링크를 통해 google `authenticator` 앱을 설치해 줍니다.
2. google authenticator 앱을 실행한 후 `QR코드`를 스캔해줍니다.
3. 한 번 발급된 인증키는 다시 찾을 수 없으니 반드시 적어두도록 합니다.
4. google authenticator 앱에 등록된 `6자리 인증코드`를 입력한 후 `인증완료` 버튼을 클릭합니다.
