# 아이디 찾기

<img width="800" src={findId} alt="아이디찾기 이미지" />

1. CS Checker 로그인 페이지에서 [아이디 찾기](https://compliance-test.cstudio.app/forgot/id)를 클릭합니다.
2. 이메일 주소를 입력한 후 `코드 전송 버튼`을 클릭합니다. 예) `cloudstudio@cloudraw.kr`
3. 입력한 이메일 주소를 통해 전송된 `여섯 자리의 인증번호`를 확인합니다.
4. 클라우드 스튜디오로 돌아와 인증번호를 입력한 후 `아이디 찾기`을 클릭합니다.
5. `다음 버튼`을 클릭합니다.
6. 이메일 주소로 전송된 아이디를 확인합니다.
