# 비밀번호 재설정

<img width="800" src={findPw} alt="비밀번호재설정 이미지" />

1. CS Checker 로그인 페이지에서 `비밀번호 재설정`를 클릭합니다.
2. 이메일 주소를 입력한 후 `코드 전송 버튼`을 클릭합니다. 예) `cschecker@cloudraw.kr`
3. 입력한 이메일 주소를 통해 전송된 `여섯 자리의 인증번호`를 확인합니다.
4. CS Checker로 돌아와 인증번호를 입력한 후 `확인`을 클릭합니다.
5. `비밀번호 재설정 버튼`을 클릭합니다.
6. `비밀번호 재설정 버튼`을 클릭합니다.
7. 새 비밀번호를 입력합니다.
8. 확인을 위해 다시한번 새 비밀번호를 입력합니다.
9. `비밀번호 재설정 버튼`을 눌러 완료합니다.
10. `비밀번호 재설정 버튼`을 눌러 완료합니다.

<p>&nbsp;</p>

:::info
비빌번호는 아래의 조건을 만족해야합니다.

-   영문 대소문자, 숫자, 특수문자만을 사용
-   시작은 영문 소문자
-   8자리 이상 20자리 이내

:::
