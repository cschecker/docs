조직관리

조직관리를 위해 조직관리>Azure Active Directory 또는 AWS Organizations로 이동합니다. 조직 정보는 크레센셜에 등록된 정보를 바탕으로 구성됩니다.
크레덴셜을 변경하여 다른 조직 정보를 불러올 수 있습니다.

 

조직원 추가​

  1. 조직원을 추가할 조직(크레덴셜)을 선택합니다.

  2. 조직원 추가 버튼을 클릭합니다.

  3. 조직원 이름, 직책을 입력한 후 상위 조직원을 선택합니다.(바로 위 조직원 선택) 이메일과 연락처를 입력한 후 추가하기 버튼을 클릭합니다.
