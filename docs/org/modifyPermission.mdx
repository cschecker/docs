# 관리자 권한 수정

1.  권한을 수정하려는 조직원에서 `Show Details` 버튼을 클릭합니다.

    <img width="200" src={org4} alt="이미지" />

2.  `관리자 권한 수정하기` 버튼을 클릭합니다.

    <img width="400" src={org5} alt="이미지" />

3.  **권한 추가:** 전체 권한 목록에서 추가하려는 권한 선택 후 오른쪽 `화살표 아이콘`을 클릭합니다.<br/>
    **권한 삭제:** 현재 권한에서 삭제하려는 권한을 선택한 후 우측 상단 `삭제버튼`을 클릭합니다.
    <img width="400" src={org6} alt="이미지" />

:::note

권한 수정시 Graph API : RoleManagement.ReadWrite.Directory 권한이 필요합니다.

:::

:::info

**권한종류:** Global Reader, Authentication Policy Administrator, Security Operator, Directory Readers, Helpdesk Administrator, Security Administrator, Global Administrator, Privileged Role Administrator
, Security Reader, Azure AD Joined Device Local Administrator

:::
