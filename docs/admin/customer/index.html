하위 조직

하위 조직을 위해 관리자>하위 조직 페이지를 클릭합니다.

정보

cloud security checker: super org(최상위 조직)이 sub org(하위 조직)을 관리할 수 있는 서비스입니다. super org(최상위 조직)는 하위에 여러 관리자를 둘 수 있습니다.

하위 조직 추가 및 삭제, 수정​

추가​

  1. 하위 조직 추가 버튼을 클릭합니다.

  2. 하위 조직명을 입력합니다.

  3. 회사 로고를 선택합니다. (196 x 60px)admin4

  4. 추가하기를 클릭합니다.

초대됨: 이메일을 발송한 상태
활성화: 초대 이메일을 받고 서비스에서 OTP까지 등록한 상태 OTP 등록 방법


 

삭제, 수정​

  1. 하위 조직 편집 버튼을 클릭합니다.

  2. 수정: 텍스트 입력 칸을 클릭하여 하위 조직명을 수정한 후 수정하기를 클릭하여 저장해줍니다.
    삭제: 삭제하기 버튼을 클릭합니다.

 

하위 조직 그룹에 유저 초대하기​

  1. 초대하기 버튼을 클릭합니다.

  2. 초대할 유저의 이름(실명)과 이메일을 입력합니다.

  3. 유저의 권한을 설정합니다(복수 선택 가능). 여러명을 초대하고 싶다면 목록에 추가하기 +을 클릭한 후 유저정보를 입력합니다.


초대됨: 이메일이 발송되었습니다.
활성화됨: 사용자가 초대 이메일을 수신하고 서비스에서 OTP를 등록했습니다. OTP를 등록하는 방법.
