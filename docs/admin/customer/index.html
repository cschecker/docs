고객사 관리

고객사 관리를 위해 관리자>고객사 관리페이지를 클릭합니다.

info

cloud security checker: super org(최상위 조직)이 sub org(고객사)를 관리할 수 있는 서비스입니다. super org(최상위 조직)는 하위에 여러 관리자를 둘 수 있습니다.

고객사 추가 및 삭제, 수정​

추가​

  1. 고객사 추가하기 버튼을 클릭합니다.

  2. 고객사명을 입력합니다.

  3. 회사 로고를 선택합니다. (196 x 60px)

  4. 추가하기를 클릭합니다.

 

삭제, 수정​

  1. 고객사 편집 버튼을 클릭합니다.

  2. 수정: 텍스트 입력 칸을 클릭하여 고객사명을 수정한 후 수정하기를 클릭하여 저장해줍니다.
    삭제: 삭제하기 버튼을 클릭합니다.

 

고객사 그룹에 유저 초대하기​

  1. 초대하기 버튼을 클릭합니다.

  2. 초대할 유저의 이름(실명)과 이메일을 입력합니다.

  3. 유저의 권한을 설정합니다(복수 선택 가능). 여러명을 초대하고 싶다면 + 버튼을 클릭한 후 유저정보를 입력합니다.
