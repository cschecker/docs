관리자 설정

정보

cloud security checker: super org(최상위 조직)이 sub org(고객사)를 관리할 수 있는 서비스입니다. super org(최상위 조직)는 하위에 여러 관리자를 둘 수 있습니다.

관리자 초대​

  1. 관리자 추가 버튼을 클릭합니다.

  2. 초대하기를 클릭하여 초대 이메일을 발송합니다.

    초대할 유저의 이름과 이메일을 입력합니다.
    목록에 추가하기 +를 눌러서 관리자 목록을 추가합니다.
    초대하기를 클릭하여 초대 이메일을 발송합니다.

초대됨: 이메일을 발송한 상태
활성화: 초대 이메일을 받고 서비스에서 OTP까지 등록한 상태 OTP 등록 방법

 

관리자 삭제​

관리자 테이블>설정(점 세 개 아이콘)을 클릭한 후 관리자 삭제 버튼을 클릭합니다. 재초대 버튼은 상태가 초대됨인 상태에서만 활성화됩니다.
초대됨: 이메일을 발송한 상태
활성화: 초대 이메일을 받고 서비스에서 OTP까지 등록한 상태 OTP 등록 방법
