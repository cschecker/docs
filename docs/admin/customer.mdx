# 하위 조직

하위 조직을 위해 `관리자>하위 조직` 페이지를 클릭합니다.

:::info

**cloud security checker:** super org(최상위 조직)이 sub org(하위 조직)을 관리할 수 있는 서비스입니다. super org(최상위 조직)는 하위에 여러 관리자를 둘 수 있습니다.

:::

## 하위 조직 추가 및 삭제, 수정

### 추가

1.  `하위 조직 추가 버튼`을 클릭합니다.

    <img width="200" src={admin4} alt="이미지" />

2.  `하위 조직명`을 입력합니다.
3.  회사 로고를 선택합니다. (196 x 60px)admin4
4.  추가하기를 클릭합니다.
    <img width="400" src={admin5} alt="이미지" />

<p>&nbsp;</p>

### 삭제, 수정

1. `하위 조직 편집 버튼`을 클릭합니다.

    <img width="200" src={admin6} alt="이미지" />

2. **수정:** 텍스트 입력 칸을 클릭하여 하위 조직명을 수정한 후 `수정하기`를 클릭하여 저장해줍니다.<br/> **삭제:** `삭제하기 버튼`을 클릭합니다.
    <img width="400" src={admin7} alt="이미지" />

<p>&nbsp;</p>

## 하위 조직 그룹에 유저 초대하기

1. `초대하기 버튼`을 클릭합니다.

    <img width="200" src={admin10} alt="이미지" />

2. 초대할 유저의 `이름(실명)`과 `이메일`을 입력합니다.

    <img width="500" src={admin8} alt="이미지" />

3. 유저의 권한을 설정합니다(복수 선택 가능). 여러명을 초대하고 싶다면 `목록에 추가하기 +`을 클릭한 후 유저정보를 입력합니다.
    <img width="500" src={admin9} alt="이미지" />
