# 로그인

<img width="400" src={login1} alt="이미지" />

1. `이메일`을 입력합니다.
2. `비밀번호`를 입력합니다.

<img width="400" src={login2} alt="이미지" />

3. OTP 인증 팝업이 뜨면 핸드폰에 설치된 Google Authenticator 앱을 열어 코드를 확인합니다.
4. 확인한 코드를 입력합니다. (인증키를 분실했다면 KISMI로 문의해주세요. 이메일: help@kismi.kr)
5. `로그인 버튼`을 클릭합니다.

[OTP 등록 방법은 여기를 참고해주세요](/docs/otp/)
