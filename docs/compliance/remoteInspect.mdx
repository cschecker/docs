# 원격진단

## 원격진단 권한

원격진단 권한 조회를 위해서는 `ⓘ 버튼`을 클릭합니다.

<img width="400" src={remoteInspect29} alt="조회하기 이미지" />
<p></p>
<img width="400" src={remoteInspect30} alt="조회하기 이미지" />
<p></p>

## 실행 방법

1. 원격진단을 위해 `점검>점검 관리>원격진단` 탭을 클릭합니다.
2. 그룹목록에서 점검하려는 `그룹을 선택`합니다.

:::info

원격진단은 선택한 그룹을 기준으로 진행됩니다. 원격진단 전 `그룹을 먼저 생성`해주세요.
[그룹생성방법](/docs/compliance/group)

:::

3. `원격진단 탭`을 클릭합니다.

    <img width="200" src={remoteInspect2} alt="이미지" />

4. `진단하기 버튼`을 클릭합니다.

    <img width="200" src={remoteInspect3} alt="이미지" />

5. 진단 제목을 입력(무엇에 관한 점검인지)한 후 `진단 시작하기` 버튼을 클릭합니다.

    <img width="400" src={remoteInspect4} alt="이미지" />

<p>&nbsp;</p>

### 점검 결과 상세

테이블 한 칼럼마다 상세 결과를 확인 할 수 있습니다. 확인하고 싶은 칼럼을 클릭합니다.

<img width="800" src={remoteInspect8} alt="이미지" />

<p>&nbsp;</p>

**상세결과 테이블**<br/>
3-1. 제목<br/>
3-2. 항목명<br/>
3-3. 상세 설명<br/>
3-4. 진단 결과<br/>
3-5. 진단 방법<br/>

<img width="800" src={remoteInspect28} alt="이미지" />

:::caution

상세정보를 수정한 후 수정사항 저장하기 버튼을 누르지 않고 페이지를 이동하면 **변경사항이 반영되지 않습니다.**

:::

## 레포트 생성

점검이 완료 되면 테이블로된 결과를 확인할 수 있습니다. 테이블에서 `영역 Scope, 항목명, 점검 기준, 중요도, 점검결과, 점검 결과 상세`의 6가지 항목을 확인할 수 있습니다.
`점검기준`에 대한 자세한 내용은 [여기](/docs/compliance/result)에서 확인할 수 있습니다.
CS Checker에서는 엑셀과 워드 두 종류의 보고서를 제공합니다.
