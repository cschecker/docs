# 점검 결과 비교

1. 점검 결과 비교는 같은 그룹 내 진단끼리만 가능합니다.(다른 그룹끼리 불가능) 먼저 비교할 대상 그룹을 선택합니다.
2. 비교날짜를 선택합니다. 선택한 날짜와 직전 점검결과를 비교합니다.
3. 비교 결과를 확인합니다.
