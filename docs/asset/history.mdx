# 자산 이력

자산 이력은 `자산을 조회한 이력`을 나타냅니다.
자산 이력에서 조회 시간, 조회자를 확인하고 원하는 이력으로 이동하거나 이력을 삭제할 수 있습니다.

1. 자산 이력 확인을 위해 `지난 자산 이력 버튼`을 클릭합니다.

<img width="200" src={assethistory1} alt="이미지" />

2. 원하는 자산 이력을 클릭하여 해당 이력을 확인할 수 있습니다. 상단으로 갈수록 최신 날짜 입니다.

<img width="400" src={assethistory2} alt="이미지" />

<p>&nbsp;</p>

## 이력삭제

삭제하려는 이력에서 `휴지통 아이콘`을 클릭합니다.
