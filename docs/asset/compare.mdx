# 자산 비교

1. 자산 비교를 위해 `자산>자산 비교`를 클릭합니다.
2. 비교할 이력을 선택합니다. `가장 최근 조회한 자산 이력`과 `내가 선택한 자산 이력`을 비교합니다.

    <img width="300" src={assetcompare1} alt="이미지" />

3. 추가된 자산과 삭제된 자산 목록을 확인합니다.

    <img width="600" src={assetcompare2} alt="이미지" />
