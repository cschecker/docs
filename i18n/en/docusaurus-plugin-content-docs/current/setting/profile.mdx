# Profile

## Profile Image

<img width="400" src={profileImage1} alt="Profile Change Image" />

1. Go to CS Checker - Settings - My Profile, and click `Change Profile`.
2. Select the desired image and apply the change.

## Resetting Password

1. CS Checker - Setting - My Profile, click `Reset Password`.

    <img width="200" src={profileImage2} alt="Password Reset Image" />

2. Enter your curent password.

    <img width="400" src={profileImage3} alt="Password Reset Image" />

3. Enter a new password.
4. Re-enter the new password to confirm.
5. Click the `Change Button` to complete the process.
