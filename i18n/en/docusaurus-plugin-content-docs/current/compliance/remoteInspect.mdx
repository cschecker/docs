# Remote Diagnostics

## How to Execute

1. Click the `Inspection > Inspection Management > Remote Diagnostics` tab to access remote diagnostics.
2. Select the `group you want to inspect` from the group list.

:::info

Remote diagnostics are conducted based on the selected group. Please `create a group first` before starting remote diagnostics.
[How to Create a Group](/docs/compliance/group)

:::

3. Click the `Remote Diagnostics tab`.

    <img width="300" src={remoteInspect2} alt="img" />

4. Click the `Diagnosis Button`.

    <img width="200" src={remoteInspect3} alt="img" />

5. Enter a diagnosis title (to specify what the inspection is about) and click the `Diagnosis Button`.

    <img width="400" src={remoteInspect4} alt="img" />

## Report Generation

Once the inspection is complete, you can view the results in a table. The table includes six columns: `Scope, Item Name, Inspection Criteria, Priority, Inspection Result, and Inspection Result Details`.
For more detailed information about `Inspection Criteria`, refer to [here](/docs/compliance/result).
CS Checker provides two types of reports: Excel and Word.
