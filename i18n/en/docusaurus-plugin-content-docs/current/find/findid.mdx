# Find ID

<img width="800" src={findId} alt="Find ID img" />

1. On the CS Checker login page, click [Find ID](.app/forgot/id).
2. Enter your email address and click `Send Code Button`. For example) `cloudstudio@cloudraw.kr`.
3. Check the `six-digit verification code` sent to the email address you entered.
4. Return to Cloud Studio, enter the verification code, and click `Find ID`.
5. Click the `Next Button`.
6. Check your email for the ID that was sent.
