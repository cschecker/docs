# Reset Password

<img width="800" src={resetPw} alt="Reset Password img" />

1. On the CS Checker login page, click [Forgot Password](/docs/find/findpw).
2. Enter your email address and click `Send Code Button`. For example) `cschecker@cloudraw.kr`
3. Check the `six-digit verification code` sent to the entered email address.
4. Return to CS Checker, enter the verification code, and click `Confirm`.
5. Click the `Reset Password Button`.
6. Enter your new password.
7. Re-enter the new password for confirmation.
8. Click the `Reset Password Button` to complete.

<p>&nbsp;</p>

:::info
Passwords must meet the following conditions.

-   Use only uppercase and lowercase English letters, numbers, and special characters.
-   Start with a lowercase English letter.
-   Be between 8 and 20 characters long.

:::
