# Customer Management

To manage customers, click on the `Admin>Customer Management`page.

:::info

**cloud security checker:** The super org (top-level organization) is a service that can manage sub orgs (customers). The super org (top-level organization) can have multiple administrators under it.

:::

## Adding, Deleting, and Editing Customers

### Adding

1.  Click the `Add Sub-Organization Button`.

    <img width="200" src={admin4} alt="img" />

2.  Enter the `Customer Name`.
3.  Select the company logo. (196 x 60px)
4.  Click the Add button.
    <img width="400" src={admin5} alt="img" />

<p>&nbsp;</p>

### Deleting, Editing

1. Click the `Edit Sub-Organization Button`.

    <img width="200" src={admin6} alt="img" />

2. **Editing:** Click the text input field, modify the customer name, and then click `Edit` to apply changes.<br/> **Deleting:** Click the `Delete Button`.
    <img width="400" src={admin7} alt="img" />

<p>&nbsp;</p>

## Inviting Users to a Customer Group

1. Click the `Invite User Button`.

    <img width="200" src={admin10} alt="img" />

2. Enter the `Name(Real Name)`and `Email` of the user you want to invite.

    <img width="500" src={admin8} alt="img" />

3. Set the user's permissions (multiple selections allowed). If you want to invite multiple users, click the `Add to List +` and enter the user information.
    <img width="500" src={admin9} alt="img" />
