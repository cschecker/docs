# Log

Logs are provided for six categories: Assets, Inspections, Users, Customers, Organization Management, and Credentials. You can specify a desired time period to view the logs.

<img width="800" src={log1} alt="img" />

To view detailed logs, click the `Magnifying Glass Icon`.

<img width="150" src={log2} alt="img" />

Additional information can be checked in the log details popup.
